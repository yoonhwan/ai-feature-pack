================================================
   데일리 트렌드 뷰어 — 사용법
================================================

유튜브 · 쇼츠 · 릴스 · 틱톡 · X(트위터) · 스레드 인기 콘텐츠와
AI 영상 모델·뉴스를 매일 한 곳에서 보는 프로그램입니다.

------------------------------------------------
■ 준비물 (딱 한 번만)
------------------------------------------------
파이썬(Python 3)이 필요합니다. 없으면 아래에서 설치하세요.
  https://www.python.org/downloads/

  · 윈도우: 설치 화면에서 "Add Python to PATH" 를 꼭 체크하세요.
  · 맥: 그냥 설치하면 됩니다.
(별도 프로그램 설치는 필요 없습니다. 파이썬만 있으면 됩니다.)

------------------------------------------------
■ 실행 방법
------------------------------------------------
  · 맥      →  "실행-Mac.command"  를 더블클릭
  · 윈도우  →  "실행-Windows.bat"  를 더블클릭

검은 창이 하나 뜨고, 잠시 뒤 브라우저에서 화면이 열립니다.
(자동으로 안 열리면 브라우저 주소창에  http://localhost:28088  입력)

종료: 그 검은 창을 닫으면 됩니다.

------------------------------------------------
■ 맥에서 "확인되지 않은 개발자" 경고가 뜰 때
------------------------------------------------
"실행-Mac.command" 를 그냥 더블클릭하면 막힐 수 있습니다.
→ 파일을 마우스 오른쪽 클릭 → "열기" → 다시 "열기" 를 누르면 됩니다.
(처음 한 번만 이렇게 하면 다음부터는 더블클릭으로 열립니다.)

------------------------------------------------
■ 탭 구성
------------------------------------------------
위쪽 탭에서 유튜브 / 쇼츠 / AI 영상 / 릴스 / X / 스레드 / 틱톡을 전환합니다.
"AI 영상" 탭만 랭킹이 아니라 신규·트렌딩 AI 영상 모델과 뉴스를 모아 보여줍니다.

------------------------------------------------
■ 보기 모드 (전체 / 등록목록 / 태그)
------------------------------------------------
탭 아래 첫 줄에서 세 가지 보기 모드를 고를 수 있습니다.

  · 전체     — 해당 플랫폼의 인기 콘텐츠를 그냥 랭킹으로 보여줌
  · 등록목록 — 내가 추가한 계정(채널)들의 콘텐츠만 모아서 랭킹
  · 태그     — 내가 등록한 태그/해시태그에 해당하는 콘텐츠만 랭킹

(스레드 탭은 태그 모드가 없고 등록목록만 지원합니다.)

계정·태그는 각 탭 화면에서 입력창에 적고 추가 버튼을 누르면 등록되고,
등록된 항목 옆의 ✕(삭제) 버튼으로 지울 수 있습니다.

------------------------------------------------
■ 정렬
------------------------------------------------
  · "정렬" 메뉴로 조회수순 / 좋아요순 전환
  · 그 옆 "▼ 내림차순" / "▲ 오름차순" 버튼으로 정렬 방향을 뒤집을 수 있음
    (많이 본 것부터 볼지, 적게 본 것부터 볼지 선택)
  · 기간 필터(오늘 / 이번 주 / 이번 달)와 개수(50~200개)도 위쪽에서 조절

------------------------------------------------
■ 검색
------------------------------------------------
  · 유튜브: 헤더의 검색창으로 키워드 검색 (조회수순 표시)
  · 릴스: 탭 안의 "#해시태그 검색"으로 즉석 검색
  · 틱톡: 탭 안의 검색창으로 즉석 검색
  · X / 스레드: 탭 안에 검색창이 있지만, 이 두 곳은 공식 무인증 검색이 없어서
    누르면 이 화면이 아니라 X·스레드 사이트 새 탭에서 검색 결과가 열립니다.

------------------------------------------------
■ 릴스(인스타그램) 탭은 로그인이 필요합니다
------------------------------------------------
릴스 데이터를 가져오려면 인스타그램 로그인 세션이 있어야 합니다.
릴스 탭을 처음 열면 안내가 뜹니다 — 순서대로 따라하면 됩니다.

  1) "인스타그램 로그인 페이지 열기" 버튼으로 새 탭에서 로그인
  2) 로그인한 그 탭에서 F12(개발자 도구) → Network(네트워크) 탭 클릭
  3) 페이지 새로고침 → 맨 위 www.instagram.com 요청 클릭
  4) 오른쪽 Headers → Request Headers 안의 Cookie 값 전체 복사
  5) 릴스 탭으로 돌아와 입력창에 붙여넣고 저장

한 번 저장해두면 한동안 유지되고, 만료되면 다시 안내가 뜹니다.

------------------------------------------------
■ 참고
------------------------------------------------
  · 인터넷 연결이 필요합니다.
  · 각자 자기 컴퓨터에서 실행하므로 데이터가 잘 나옵니다.
  · 틱톡 탭은 처음 열 때 썸네일이 다 뜨는 데 몇십 초 걸릴 수 있어요.
  · 스레드 탭은 실시간 조회가 막히면 계정 바로가기로 안내합니다.
  · 결과는 1시간 캐시되며, 우측 상단 "새로고침"으로 즉시 갱신할 수 있어요.
