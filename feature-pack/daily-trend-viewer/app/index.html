인스타그램 웹 내부 API를 통해 등록된 계정들의 최신 릴스를 가져와 조회수순으로 정렬합니다. 아래에서 즐겨보는 크리에이터 계정을 자유롭게 추가·삭제하세요.

1) 아래 버튼으로 인스타그램 로그인 페이지를 새 탭에서 열고 로그인하세요
2) 로그인된 그 탭에서 F12 키를 누르세요 (안 열리면 우측 상단 ⋮ 메뉴 → 도구 더보기 → 개발자 도구)
3) 개발자 도구 상단 탭에서 Network(네트워크)를 클릭하세요
4) 브라우저 새로고침 버튼(또는 F5)으로 페이지를 새로고침하세요 — 목록에 요청들이 쭉 뜹니다
5) 목록 가장 위쪽에 있는 www.instagram.com 요청을 클릭하세요
6) 오른쪽에 뜨는 패널에서 Headers 탭 → Request Headers 항목을 찾고, 그 안의 Cookie 값 전체를 복사하세요
7) 아래 입력창에 붙여넣고 저장을 누르세요
