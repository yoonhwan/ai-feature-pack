# Feature Pack: auto (AutoResearch)

Karpathy AutoResearch 패턴 기반 자율 실험 루프 스킬.

> 에이전트가 **타깃 파일 1개를 수정 → 실행 → 평가 → commit/revert**를 반복하며 자율적으로 최적화한다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| **베이스라인 수립** | 현재 코드 그대로 실행하여 기준 점수 확립 |
| **목표 설정 인터뷰** | 사용자와 개선 목표·제약 조건 합의 |
| **자율주행 실험** | 가설→수정→실행→평가→keep/revert 반복 루프 |
| **전략 피벗** | 연속 5회 실패 시 자동으로 탐색 방향 전환 |
| **git 기반 추적** | 모든 실험을 전용 브랜치에 commit, 실패 시 reset |
| **결과 리포트** | 점수 추이, 성공/실패 분석, 다음 단계 제안 |

## 3단계 HIL (Human-in-the-Loop)

```
Stage 1: 베이스라인  →  사용자 확인  →
Stage 2: 목표 설정   →  사용자 확인  →
Stage 3: 자율주행     →  (사용자가 멈출 때까지)
```

## 빠른 시작

```bash
# 전체 흐름 (setup → baseline → 목표 → 자율주행)
/auto

# 2시간 자율주행
/auto 2h

# 기존 브랜치에서 이어서
/auto resume

# 결과 리포트만 생성
/auto report
```

## 요구사항

| 항목 | 필수 | 비고 |
|------|------|------|
| git | ✅ | 실험 브랜치·commit·revert |
| python3 | ✅ | 문법 검증·결과 파싱 |
| API 키 | ❌ | 불필요 |
